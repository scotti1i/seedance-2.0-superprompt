# Template Index

Verified, production-grade Seedance 2.0 prompt templates. Each one was run
on at least one Seedance client and the success rate is recorded.

To use:
1. Find a template matching your scenario
2. Open the file, copy the prompt
3. Substitute the `${variables}`
4. Paste into your Seedance client (jimeng / capcut / volcengine / etc.)

To contribute: see `CONTRIBUTING.md` in repo root, use `TEMPLATE.md` as a
starting point.
